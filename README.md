### Prerequisites
* [Alex](https://www.haskell.org/alex/)
* [Happy](https://www.haskell.org/happy/)

 <br />

The plugins can be installed using [Cabal](https://www.haskell.org/cabal/) :
* Alex
  ```sh
    cabal install alex
  ```
* Happy
  ```sh
    cabal install happy
  ```
